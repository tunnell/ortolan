Mathematical Background

Chi-Square Goodness-of-Fit (Dice)

Measures whether observed frequencies differ significantly from expected frequencies. For a fair die, each face should appear with probability 1/6.

χ² = Σ (Oᵢ - Eᵢ)² / Eᵢ

With 6 faces, we have 5 degrees of freedom. The critical value at α = 0.05 is 11.07.

Binomial Test (Coins)

Tests whether the proportion of heads differs significantly from 0.5. Uses the exact binomial distribution for small samples and normal approximation for larger ones.

p-value = 2 × P(X ≥ |observed - expected|)

Runs Test (Independence)

Detects non-random patterns in a sequence. A "run" is a consecutive sequence of the same outcome (e.g., HHH or TT).

E[R] = (2n₁n₂)/(n₁+n₂) + 1
σ²[R] = (2n₁n₂(2n₁n₂-n₁-n₂)) / ((n₁+n₂)²(n₁+n₂-1))

A very low number of runs suggests clustering; a very high number suggests alternation. Both indicate non-randomness.

Wilson Confidence Interval

More accurate than the normal approximation for proportions, especially with small samples or extreme proportions:

p̂ᵤₚₚₑᵣ = (p̂ + z²/2n + z√(p̂(1-p̂)/n + z²/4n²)) / (1 + z²/n)

How Many Rolls/Flips Do I Need?

I recommend ~100. This provides good statistical power for detecting meaningful bias.

  • ~60: Detect gross bias (30%+ deviation)
  • ~100: Good power for moderate bias (recommended)
  • ~300+: Detect subtle bias (~20% deviation)
